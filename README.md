## Razlaga

Ustvarjanje tabele :

```html
<table></table>
```

Ustvarjanje stolpcev  v tabeli :

```html
<tr></tr>
```
Ustvarjanje vrstic v tabeli :

```html
<td></td>
```
### Uporabljeni input type-i

Input za tekst :

```html
<input type="text" name="ime">
```
![alt text]()

Input za izbor možnosti (radio) :

```html
<input type="radio" name="spol" value="moški">
```
![alt text]()

Input za izbor možnosti (select) :

```html
<select required>
<option value="Slovenija">Slovenija</option>
```
![alt text]()

Input za izbor možnosti (checkbox) :

```html
<input type="checkbox" value="predavanje">Ali so IPv6 tuneli varnostna grožnja</br>
```
![alt text]()

Input za večjo količino teksta :

```html
<textarea style="height:60px">Ni opomb.</textarea>
```
![alt text]()

Gumb za odposlati svojo izbiro :

```html
<input type="text" name="geslo" style="width:100px">
```
![alt text]()

### Dodatne stvari

 - required (uporabljamo takrat kadar je nujno, da uporabnik izbere stvar pri nekem inputu);
```html 
<input type="text" name="ime" required>
```
